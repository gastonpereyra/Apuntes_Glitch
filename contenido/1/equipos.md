# Equipos

<img src="https://cdn.glitch.com/d5cdac21-5e23-4601-8243-d5645f80aa71%2Fintro.svg?1538169918301" width="900">

Se pueden armar equipos y codear juntos.

## Limitaciones, o no tanto

Por ahora esta en Beta, asi que el limite aun no se aplica.

* 1 equipo = Ilimitados User = 5 proyectos gratis, luego pagar.

## Formar un Equipo

Muy sencillo, entre las opciones de usuario se puede clickear en el boton `Create Team`

<img src="https://github.com/gastonpereyra/Apuntes_Glitch/blob/master/imagenes/Glitch_team_1.png" width="900">

Se le asigna un nombre

<img src="https://github.com/gastonpereyra/Apuntes_Glitch/blob/master/imagenes/Glitch_team_2.png" width="900">

Y se empieza.

## Edición en Tiempo-Real

<img src="https://github.com/gastonpereyra/Apuntes_Glitch/blob/master/imagenes/Glitch_team_3.png" width="450">

Todos pueden codear al mismo tiempo y ver que estan haciendo.

## Control de Versión

<img src="https://github.com/gastonpereyra/Apuntes_Glitch/blob/master/imagenes/Glitch_team_4.png" width="900">

Cuaquiera puede volver a un estado anterior usando la interfaz grafica de Git que posee Glitch, sin conocer Git incluso.

## Hosting automatico

<img src="https://github.com/gastonpereyra/Apuntes_Glitch/blob/master/imagenes/Glitch_team_5.png" width="900">

El Hosting y el deployment de la app se hace de manera automatica.

## Privado

<img src="https://github.com/gastonpereyra/Apuntes_Glitch/blob/master/imagenes/Glitch_team_6.png" width="900">

Se pueden generar Apps privadas, que solo el equipo puede acceder.

## Analytics

<img https://cdn.glitch.com/d5cdac21-5e23-4601-8243-d5645f80aa71%2Fanalytics-wide2.png?1538433070240 width="900">

Glitch otorga una herramienta con datos y estadisticas de las apps.
